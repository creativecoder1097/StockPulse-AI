# ⚡ StockPulse AI - Gamified Quantitative Terminal

StockPulse AI is a high-performance, beginner-friendly stock market analysis tool. It combines institutional-grade quantitative metrics with a sleek, gamified "Neon-Dark" interface.

## 🚀 Features
- **AI Pulse Discovery:** 7-day price forecasting using Random Forest Ensembles.
- **Monte Carlo Simulations:** 300+ simulated future paths to visualize probability.
- **Risk Profiling:** Automatic calculation of Alpha, Beta, Sharpe Ratio, and VaR.
- **Hypothesis Lab:** Statistical testing for Market Normality and Stationarity.
- **Beginner Friendly:** Includes tooltips and a plain-English glossary for complex terms.

## 🛠️ Tech Stack
- **Language:** Python 3.9+
- **Framework:** Streamlit (UI)
- **Math/ML:** Scipy, Scikit-Learn, Numpy, Pandas
- **Graphics:** Plotly (Interactive Charts)
